# Little Feather 4D5F Sales Platform API

API para Sales Platform usando **Cloudflare Worker**.

## Endpoints de ejemplo

- `GET /` → "Hello World from Worker API!"
- `GET /sales` → Devuelve datos de ventas en formato JSON

## CI/CD

El repo está conectado con **GitHub Actions** para publicar automáticamente el Worker en Cloudflare cada vez que se haga push a `main`.
